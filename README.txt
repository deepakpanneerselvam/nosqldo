nosqldo
=======

NoSql DataObject encapusulate the CRUD operation of all noSQL datastore.

Simplifies the need for CRUD operation in generic way and based on configuartion driven approach.

Any change of new parameter can be directly configured in the dataTemplate schema.

Uses the concept of Java Binding ,so  the input can be directly xml or Object.

Unit tested CRUD operations of MongoDB ,Cassandara and Redis .Not Hbase.
